# zyhx-spider-ZGS-FLS
智宇慧轩爬虫，基于 Python+Tkinter 开发的 GUI 人民日报数据库文献采集工具，仅限浙江工商大学在校师生凭图书馆校内合法授权 IP使用，用于个人学术科研学习。可自定义检索关键词、起止日期、抓取页数，支持正文去冗余清洗，抓取结果自动拆分保存：文章元数据存入outline.txt、正文内容存入content.txt。软件启动自带法律免责弹窗，遵循《著作权法》《网络安全法》等法规，严禁商用、越权非法批量爬取；需先行跳转校图书馆官网完成数据库授权访问，站点改版可能造成解析异常，数据有效性由使用者自行核验。目前仅限人民日报数据爬取，作者未来考虑增加更多的爬虫功能
